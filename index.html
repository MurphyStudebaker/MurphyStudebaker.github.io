Cascara

  • MySQL
  • Android
  • REST API

A mobile application that uses crowd-sourced data to help users find the perfect coffee house in their area. I populated a seven table MySQL database, built a RESTful API, and designed and built an Android application using the Volley library.

Petestrian

  • Mobile
  • UX Design

A mobile application that encourages users to lead more active lifestyles by emotionally pairing their step count to the health of a digital pet. I researched current academic literature on health-related mobile applications, conducted user testing, and built both low-fidelity and high-fidelity prototypes.
